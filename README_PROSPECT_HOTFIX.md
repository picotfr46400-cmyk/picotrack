# Invitation utilisateur Supabase

Utilise la version finale : `README_DEPLOIEMENT_FINAL.md`.

Secrets Edge Function à créer dans Supabase :

```txt
URL
ANON_KEY
SERVICE_ROLE_KEY
```

Ne pas utiliser les noms `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY` dans l'interface Supabase, car Supabase refuse le préfixe `SUPABASE_`.
